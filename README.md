## ğŸš€ Quick Start

### **For Developers**
```bash
# 1. Clone the repository
git clone https://github.com/Pranay50x/PortReview.git
cd PortReview

# 2. Set up environment variables
cp frontend/.env.example frontend/.env
# Add your API keys (GitHub OAuth, Google OAuth, Gemini API)

# 3. Install dependencies
# Frontend
cd frontend && npm install

# Backend
cd ../backend && pip install -r requirements.txt

# 4. Start the services
# Backend (Terminal 1)
cd backend && uvicorn app.main:app --reload

# Frontend (Terminal 2) 
cd frontend && npm run dev
```

### **For Users**
1. **Visit** [PortReview.com](http://localhost:3000)
2. **Login** with GitHub
3. **Click** "Auto-Generate Portfolio"
4. **Wait** 60 seconds â±ï¸
5. **Share** your professional portfolio! ğŸ‰
